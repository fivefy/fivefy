PG 테스트 센터

포인트 충전, 환불, 빌링키, 자동결제 스케줄러 테스트를 한 화면에서 실행합니다.

포인트 충전

  1. 로그인합니다.
  2. 구독 화면에서 Store ID / Channel Key를 저장합니다.
  3. 상품을 선택하고 결제 요청을 누릅니다.
  4. PortOne 결제 완료 후 웹훅 처리 뒤 지갑을 조회합니다.

자동결제 테스트

  1. 빌링키용 Store ID / Channel Key를 입력합니다.
  2. 카카오페이 또는 카드 빌링키를 등록합니다.
  3. 카드 청구 테스트로 포인트를 충전합니다.
  4. 결제일 당기기 후 구독료 차감 테스트를 실행합니다.

프로젝트 전체 API

왼쪽에서 기능을 선택하고, 오른쪽의 경로/본문을 상황에 맞게 수정한 뒤 실행하세요.

기본 사용법

  1. 먼저 로그인해서 JWT를 저장합니다.
  2. 목록에서 API를 선택합니다.
  3. {trackId} 같은 path variable을 실제 ID로 바꿉니다.
  4. 필요하면 JSON Body를 수정하고 실행합니다.

권한/파일 업로드

  1. /api/admin/**는 ADMIN 권한 토큰이 필요합니다.
  2. 트랙 신청의 multipart API는 request JSON과 audioFile을 같이 보냅니다.
  3. SSE API는 브라우저 스트림 응답을 텍스트로 표시합니다.
  4. 웹훅은 PortOne 서버가 호출하는 API라 수동 실행은 테스트용입니다.
